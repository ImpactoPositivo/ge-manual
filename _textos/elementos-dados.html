Visão geral

Elementos de dados são usados para inserir dados em um processo ou dados de saída de um processo. No BPMN, os dados podem ser modelados por vários tipos de elementos de dados, como objetos de dados, entradas de dados, saídas de dados e armazenamentos de dados. Os dados também podem ser gerenciados pelo estado do objeto, como instanciado, concluído, excluído, etc. Por exemplo, no processo Relatório de Despesas, um recibo carregado como um arquivo de Documento de Entrada é um objeto de dados de entrada e uma fatura gerada como um Documento de Saída é um objeto de dados de saída. Os dados gerados a partir de Formulário Dinâmicos e variáveis são salvos em uma tabela de relatório e representados como um armazenamento de dados no mapa do processo.

Tipos de dados

Altere o tipo de objeto de dados para "Entrada", "Saída" ou "Vazio" no menu de contexto do elemento de objeto de dados:

  • Entrada: Selecione este tipo de objeto de dados para indicar que a entrada será de um aplicativo externo.
  • Saída: Selecione este tipo de objeto de dados se os dados do processo forem usados para gerar um documento que será exportado do Gestor Express.
  • Vazio: Selecione este tipo de objeto de dados se os dados não forem de entrada ou saída.
