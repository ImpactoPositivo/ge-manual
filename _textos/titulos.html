Visão geral

Títulos e subtítulos são usados para agrupar campos sob títulos e fornecer informações ao usuário sobre a finalidade do formulário.

Ao visualizar um Formulário Dinâmico ou executar um caso, um controle de título tem a seguinte aparência:

Exemplo de controle de título

Para este exemplo, adicione um controle "Título" arrastando-o e soltando-o no modelo de Formulário Dinâmico.

Clique em qualquer espaço vazio dentro do controle e suas propriedades serão exibidas no painel esquerdo. Primeiro, a propriedade "tipo" fornece o nome do controle e não pode ser alterada.

A propriedade "id" é o campo e identificador Html que pode ser usado com código Javascript. Use o nome da propriedade "id" para referência. Clique no espaço cinza do Formulário Dinâmico como visto na imagem abaixo e as propriedades do Formulário Dinâmico completo aparecerão no painel do lado esquerdo.

Clique no botão "editar" na propriedade "javascript" e um editor será aberto.

Adicione o seguinte código no editor:

jQuery('#title0000000001').setLabel('Gestor Express');

Este exemplo de código obtém o objeto jQuery para um campo com o ID "title0000000001". Em seguida, ele chama o método control.setLabel() do objeto para alterar o rótulo para "Gestor Express". O resultado é mostrado na imagem abaixo.

A propriedade a seguir para o controle é a propriedade "rótulo" que exibe o texto escrito no título. Leve em consideração que, se o código Javascript foi usado, qualquer texto escrito posteriormente não terá nenhum efeito no controle, portanto, é recomendável apagar o código javascript escrito anteriormente para que a propriedade "rótulo" tenha efeito. Agora, tente escrever "EXEMPLO DE TÍTULO" e vá até a visualização para observar o resultado deste texto.

Propriedades

Títulos e subtítulos têm as seguintes propriedades:

PropriedadeDescrição
Modelotitle/subtitle(somente leitura)
ID[Requeridos]
Identificador HTML exclusivo
RótuloRótulo do título/subtítulo

Depois de adicionar um título ou subtítulo usando a Ferramenta de Desenho do formulários, clique no controle para expandir suas propriedades no painel esquerdo. A propriedade "Tipo" indica o tipo de controle, que no caso de controle de título será "título" e no caso de subtítulo será "subtítulo".

A propriedade "Id" define o identificador exclusivo do controle selecionado. Por padrão, quando um título é adicionado, o identificador é definido como "títuloxxxxxxxxxxxx", onde os x representam o número correlativo do controle dentro do modelo. Isso significa que se um controle de título for o primeiro a ser adicionado, o id será definido como "title0000000001". Leve em conta as mesmas considerações ao trabalhar com subtítulos. A propriedade "Nome" permite definir um nome para o controle selecionado.

Essas duas propriedades permitem trabalhar com o nome e o ID ao adicionar Javascript ao formulário.

JavaScript em títulos e subtítulos

Para saber como manipular controles de Formulário Dinâmico usando JavaScript, consulte JavaScript em Formulários Dinâmicos .

Títulos e subtítulos no Gestor Express têm os seguintes componentes de campo:

  • Rótulo: O texto no título/subtítulo, que é exibido ao usuário.
  • Valor: o valor é o mesmo do rótulo .

Métodos JavaScript

Alguns dos métodos JavaScript para manipular títulos e subtítulos incluem:

MétodoDescrição
jQuery("#fieldID").getValue()Retorna o texto do título ou subtítulo.
jQuery("#fieldID").getLabel()Retorna o texto do título ou subtítulo.
jQuery("#fieldID").setValue("novoValor")Define o texto do título ou subtítulo.
jQuery("#fieldID").setLabel("novoRotulo")Define o texto do título ou subtítulo.

Para ver exemplos de código para esses métodos, consulte Funções e métodos JavaScript.

Observação: Chamar setLabel() ou setValue() para obter um título ou subtítulo irá redefinir suas propriedades e remover quaisquer propriedades personalizadas definidas pelo JavaScript. Após chamar setLabel() ou setValue(), essas propriedades precisarão ser redefinidas.
