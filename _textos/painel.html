Visão geral

Como a Ferramenta de Desenho do Formulário Dinâmico é baseado em HTML5, CSS3, Bootstrap e jQuery, um controle introduzido para Formulários Dinâmicos no Gestor Express é o painel. Um painel de controle adiciona uma funcionalidade extra e poderosa aos formulários, suportando qualquer código HTML e HTML5 inserido dentro de sua propriedade "conteúdo". Para adicionar esse controle ao modelo, arraste e solte o ícone a seguir da barra de ferramentas de controle para a Ferramenta de Desenho do Formulário Dinâmico.

Este controle está relacionado à propriedade "Biblioteca Externa" do formulário, que permite adicionar bibliotecas externas e adicionar mais funcionalidades ao controle. Dependendo das necessidades do projeto, podem ser inseridos elementos como vídeos, mapas, funções de layout jquery, css, javascript, arquivos jquery, etc.

Inserindo Variáveis no Conteúdo de um Painel

Se for necessário inserir uma variável na propriedade de conteúdo de um painel, somente uma variável poderá ser usada e nenhum outro texto ou variável poderá ser incluído.

Por exemplo, um conteúdo como:

‹p›‹b›Nome:‹/b› @@preNome‹br›‹b›Sobrenome:‹/b›@@sobreNome‹/p›

não irá inserir as variáveis corretamente no texto. Em vez disso, crie o seguinte gatilho no processo que definirá uma única variável a ser usada no painel:

//defina o conteúdo padrão:
@@conteudoPainel = '<p><b>Nome:</b><br><b>Sobrenome:</b></p>';
if (isset(@@Nome) isset(@@sobreNome)) {
   @@conteudoPainel = "<p><b>Nome:</b> ".@@Nome."<br>\n".
        "<b>Sobrenome:</b>".@@sobreNome."</p>";
}

Defina o gatilho acima para ser executado antes do Formulário Dinâmico que contém o controle do painel. Em seguida, defina a propriedade de controle do painel para conter:

@@conteudoPainel

O único problema com essa abordagem é que os gatilhos não são disparados antes e depois de um formulário dinâmico quando ele é aberto por um supervisor de processo. Se um supervisor de processo precisar acessar o formulário, é recomendável definir o gatilho para disparar durante o roteamento na tarefa anterior, para garantir que avariável @@conteudoPainel seja definida antes que o supervisor abra o formulário.

Isso pode ser um problema, no entanto, se o valor das variáveis @@Nome e @@sobreNome forem definidas na mesma tarefa que o formulário dinâmico que contém o controle do painel. Nessa situação, é recomendável usar JavaScript em vez de um gatilho para definir o conteúdo do painel de controle.

Adicione dois controles ocultos ao formulário dinâmico associado às variáveis "Nome" e "sobreNome". Em seguida, adicione o seguinte código JavaScript ao Formulário Dinâmico para definir o conteúdo do painel que neste exemplo possui o ID "nomeInfo":

var nome = $("#Nome").getValue();
var snome = $("#sobreNome").getValue();
$("#nomeInfo").html('<p><b>Nome:</b> '+nome+'<br>\n<b>sobreNome:</b>'+snome+'</p>');

Incorporando um mapa do Google usando um painel

Conforme explicado anteriormente, o conteúdo dos painéis suporta código HTML/HTML5 que inclui a iframetag. Vamos usar essa tag para renderizar um Google Map em nosso painel, essas são as etapas para conseguir isso.

  • Crie um novo formulário dinâmico, abra-o e adicione um único controle "Painel" ao modelo.
  • Clique no controle e vá para suas propriedades. Clique no editar...botão " " da propriedade "Conteúdo" e adicione o seguinte código no editor que está aberto.

Incorporando vídeos em painéis

Para este exemplo, vamos usar um iframe para incorporar um vídeo do Vimeo para que nosso formulário dinâmico inclua uma seção como a seguinte:

  • Vamos criar um novo formulário dinâmico, abri-lo e adicionar um controle de título e um controle de painel no design.
  • Clique no controle de título e vá para suas propriedades. Na propriedade "rótulo" adicione o texto "Este é o vídeo". Será o título do Formulário Dinâmico.
  • Agora, clique no painel e vá para suas propriedades no lado esquerdo. Clique no botão "editar..." e adicione o seguinte código na janela do editor que é aberta:
    • <iframe src="https://player.vimeo.com/video/121840700" width="500" height="207" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen></iframe>
  • Salve o formulário e tente renderizá-lo em "Visualizar".

Exemplo simples de controle de painel

Para este exemplo, adicione um controle "painel" arrastando-o e soltando-o do painel Controles da Web. O resultado da adição deste controle é mostrado na imagem abaixo.

Agora clique em qualquer espaço vazio dentro do controle e suas propriedades serão exibidas no painel esquerdo.

A propriedade "id" é o identificador exclusivo do controle. Altere apenas se o código HTML estiver envolvido.

A propriedade "conteúdo" abrirá uma janela do editor que adiciona o código HTML renderizado assim que o painel também for renderizado. Para isso, clique no botão "editar" como pode ser visto na imagem abaixo.

Depois que o editor de conteúdo for aberto, adicione o seguinte código:

<iframe src="https://player.vimeo.com/video/121840700" width="500" height="207" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen></iframe>

Observe que a fonte no código é um link para um vídeo que será exibido quando o controle for renderizado

A propriedade final é a propriedade "borda" que define o tamanho da borda no controle. Para este exemplo, adicione uma borda perceptível de 10px. Observe que quando o controle for renderizado a borda ficará muito mais visível do que antes.
