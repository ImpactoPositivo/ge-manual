Visão geral

Um controle de rádio (grupo) oferece apenas uma opção binária de ativação/desativação. É redondo e um ponto preto aparece no meio quando ativado (on). Um controle de grupo de rádio é usado em vez de uma caixa de seleção ou controle de grupo de seleção, quando o usuário deve selecionar apenas uma opção de um domínio de opções. Após adicionar uma caixa de seleção na ferramenta de desenho do, é possível personalizar sua funcionalidade e comportamento, utilizando o painel de propriedades.

As propriedades definidas por padrão são as seguintes:

  • tipo Definido por padrão como "radio". Esta definição não pode ser modificada.
  • id Definido por padrão como radioVar0X, onde "X" representa a numeração correspondente do controle no projeto. Esta numeração começa em 1. Após uma variável ser relacionada a este controle, o ID muda para o nome da variável, porém é possível definir um novo ID para trabalhar com o controle.
  • rótulo Definido por padrão como "radio_X" onde "X" representa a numeração correspondente do controle no desenho. Esta numeração começa em 1.
  • modo de exibição Definido por padrão como "mestre". Consulte esta documentação para obter mais informações sobre os modos disponíveis para este controle.
  • fonte de dados Definido por padrão como "banco de dados". Consulte esta documentação para obter mais informações sobre a fonte de dados disponível para esse controle.
  • Conexão de BD Definido por padrão como "PM Database". Leve em consideração que se nenhuma consulta SQL for definida na propriedade "sql", nenhuma execução será feita. Além disso, quando uma variável está relacionada a este controle que possui sua própria definição da conexão com um banco de dados (um banco de dados externo ou banco de dados do Gestor Express) esta propriedade herda a definição da variável relacionada no controle.

    Exemplo de controle de grupo de rádio

    Para este exemplo, adicione um controle "rádio" arrastando e soltando-o do painel esquerdo onde estão os "Controles da Web", na Ferramenta de Desenho do Formulário Dinâmico.

    Imediatamente após adicionar o controle, a janela Criar variável aparecerá. Para este exemplo, crie uma variável chamada "pagamento" e use a imagem abaixo como orientação para adicionar mais configurações ao controle.

    Clique em qualquer espaço vazio do controle para exibir suas propriedades no painel esquerdo. Observe que "variável", "Tipo de dados variável" e "id" serão alterados dependendo das especificações adicionadas na janela "Criar Variável".

    A propriedade "rótulo" pode ser alterada para que, ao executar um caso, ele não tenha um nome como "radio_1" e, em vez disso, possa ser nomeado (para este exemplo) "Pagamento:"

    Para a próxima propriedade que é "valor padrão", o valor é definido por padrão quando o valor é nulo. O valor padrão deve ser definido no domínio dos valores definidos nas opções ou SQL. ex: "VNZ". Se o valor padrão não estiver no domínio, o valor padrão não será definido. Se o valor padrão não for definido, nenhuma das opções será selecionada inicialmente.

    A seguir está a propriedade "Dica" que será usada para mostrar a ajuda. Por exemplo, adicione o seguinte texto: "Escolha um". Ao executar um caso, um ícone aparecerá no canto direito do controle e quando o ponteiro do mouse passar sobre ele, o texto adicionado será exibido.

    A propriedade "obrigatório" adiciona um asterisco no rótulo para indicar que o campo é obrigatório e que uma opção deve ser selecionada. Se tentar enviar o formulário, o campo mudará imediatamente para a cor vermelha e, como visto na imagem abaixo, se nada for selecionado, uma mensagem de erro será exibida.

    A próxima propriedade é a propriedade "modo de exibição" que possui muitas opções diferentes, para saber mais sobre elas passe o ponteiro do mouse sobre o ícone do ponto de interrogação, esta propriedade terá o seguinte efeito no controle:

    Mestre

    Editar

    Visualizar

    Desativado

    A propriedade "Conexão ao BD" é utilizada para selecionar entre a lista de conexões de banco de dados definidas nos objetos de processo, neste caso um PM_Database (um banco de dados do Gestor Express). A propriedade "sql" usa uma consulta para chamar valores do banco de dados. Para o uso de uma consulta sql o exemplo será alterado. O nome do controle agora será "país" para que a consulta preencha o controle com os países em uma tabela ISO_PAIS. Clique nas reticências sublinhadas ... da propriedade "sql" para abrir um "Editor SQL" e adicione o seguinte código:

    SELECT ic_uid , ic_name FROM ISO_PAIS

    Observação: O identificador * não é suportado em consultas SQL.

    JavaScript em grupos de rádio

    Para saber como manipular os controles de Formulário Dinâmico usando JavaScript, consulte JavaScript em Formulário Dinâmicos .

    Os grupos de rádio no Gestor Express têm os seguintes componentes de campo:

    • Rótulo: O texto exibido acima ou à esquerda do campo para identificá-lo ao usuário.
    • Texto: O texto (rótulo) exibido para a opção selecionada.
    • Valor: O valor armazenado (chave) para a opção selecionada.

    Métodos JavaScript

    Alguns dos métodos JavaScript para manipular grupos de rádio incluem:

    MétodoDescrição
    jQuery("#fieldID").getControl()Use esta função para obter o campo de entrada do controle, em vez do seu DIV que é obtido com $("#id")
    jQuery("#fieldID").disableValidation()Desabilite a validação do campo, para que não verifique se o campo é obrigatório quando o formulário for enviado.
    jQuery("#fieldID").enableValidation()Habilite a validação do campo, assim verificará se o campo é obrigatório quando o formulário for enviado.
    jQuery("#fieldID").getValue()Retorna o valor (chave) da opção selecionada do grupo de rádio.
    jQuery("#fieldID").getText()Retorna o texto exibido (rótulo) da opção selecionada do grupo de rádio.
    jQuery("#fieldID").getLabel()Retorna o rótulo do campo, que é o texto exibido acima ou à esquerda do campo para identificá-lo.
    jQuery("#fieldID").setValue("novo valor")Seleciona uma opção no grupo de rádio especificando seu valor (chave).
    jQuery("#fieldID").setText("novoTexto")Seleciona uma opção no grupo de rádio especificando seu texto exibido (rótulo).
    jQuery("#fieldID").setLabel("novoRotulo")Altere o rótulo do campo, que é o texto exibido acima ou à esquerda do campo para identificá-lo.
    jQuery("#fieldID").setOnchange(function(newVal, oldVal){...})Defina um manipulador de eventos de alteração, que é uma função personalizada que é executada após o valor do campo ser alterado. A função pode conter os seguintes parâmetros:
    • newVal: O novo valor que acabou de ser definido no campo.
    • oldVal: O valor antigo que foi alterado.

    Para ver exemplos de código para esses métodos, consulte Funções e métodos JavaScript.

    Exibindo opções em uma lista horizontal

    As opções em um grupo de rádio são exibidas como uma lista horizontal quando exibidas no modo de visualização ou ao executar um caso. Para grupos de rádio que possuem um grande número de opções, isso pode ocupar muito espaço e forçar o usuário a percorrer um longo formulário. Para mudar de uma lista vertical de opções para uma lista horizontal, adicione a classe "radio-inline" às DIVs ao redor de cada botão de opção.

    Por exemplo, o código JavaScript a seguir pode ser adicionado a um formulário dinâmico para alterar um grupo de rádio com o ID "selectPáises" para exibir uma lista horizontal de opções:

    $("#selectPáises").find("div.radio").addClass("radio-inline");

    Ocultar/Mostrar campos com base na seleção de rádio

    Javascript pode ser usado para ocultar ou mostrar campos no formulário, com base na seleção em um botão de rádio. Use o método setOnchange() para definir o manipulador de eventos para mostrar/ocultar os campos.

    Por exemplo, um formulário tem um botão de opção com o ID "tipodeServico" que contém as opções: "buffet", "corrio", "contabilidade", "consultoria". Se a opção "buffet" for selecionada, uma caixa suspensa com o ID "selectComida" deverá ficar visível. Caso contrário, este campo deve ficar oculto.

    O código JavaScript a seguir oculta ou mostra a lista suspensa "selectComida":

    $("#selectComida").hide(); //Ocultar quando o formulário carrega


    $("#tipodeServico").setOnchange( function(newVal, oldVal) {

       if (newVal == "buffet") {

         $("#selectComida").show();

       }

       else {

         $("#selectComida").hide();

       }

    });

    Ao usar setOnchange(), lembre-se de verificar o valor da opção selecionada no botão de opção, não em seu rótulo .
