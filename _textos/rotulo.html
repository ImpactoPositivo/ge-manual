Propriedades do Rótulo

Depois de adicionar um rótulo ao projeto, defina o texto que representa seu conteúdo. As propriedades deste controle são as seguintes:

PropriedadeDescrição
Tiporótulo(somente leitura)
IDIdentificador exclusivo de controle (definido por padrão como label000000000X onde X representa a numeração dos controles de rótulo adicionados no projeto) que pode ser alterado
TextoConteúdo do Rótulo

Observação: Siga o link das propriedades para aprender a trabalhar com cada uma delas.

Texto

O texto definido neste campo textarea será o conteúdo do comentário.

Exemplo de controle de rótulo

Para este exemplo, arraste e solte um controle "rótulo" na tela do Formulário Dinâmico.

Clique em qualquer espaço vazio dentro do controle para exibir suas propriedades no painel lateral esquerdo.

A propriedade "id" é definida por padrão, mas pode ser usada com código Javascript para adicionar funcionalidade adicional ao controle. Por exemplo, clique na borda cinza do formulário dinâmico para exibir as propriedades do formulário dinâmico no painel esquerdo. Clique no botão "editar" na propriedade "javascript". Isso abrirá um editor de JavaScript.

Uma vez no editor, adicione o seguinte código para retornar um valor específico:

jQuery('#label0000000001').setLabel('www.google.com');

Observe que, como resultado desse código, o texto do rótulo foi modificado para 'www.google.com'.

A propriedade final neste controle é a propriedade "texto" que se tornará seu conteúdo. Por exemplo, tente adicionar um texto semelhante a uma política de "Termos e Condições". Além disso, pode ser adicionada uma caixa de seleção para que o usuário possa clicar nela se concordar com as condições lidas no controle "rótulo".

JavaScript em rótulos

Para aprender a manipular controles de formulários dinâmicos usando JavaScript, consulte JavaScript em formulários dinâmicos.

Os controles de rótulo no Gestor Express têm os seguintes componentes de campo:

  • Rótulo: O texto no rótulo, que é exibido para o usuário.
  • Valor: O mesmo que o rótulo.
  • Texto: O mesmo que o rótulo.

Métodos JavaScript

Alguns dos métodos JavaScript para manipular controles de rótulos incluem:

MétodoDescrição
jQuery("#campoID") . getValue()Retorna o texto do rótulo.
jQuery("#campoID") . getText()Retorna o texto do rótulo.
jQuery("#campoID") . getLabel()Retorna o texto do rótulo.
jQuery("#campoID") . setValue("novo valor")Define o texto do rótulo.

Para ver exemplos de código para esses métodos, consulte Funções e métodos JavaScript.

Observação: Se precisar incluir o código HTML no rótulo, procure um ❬p❭ dentro do div do rótulo com a classe "geformdin-controle-rotulo" e defina o conteúdo do ❬span❭ interior com a função html(). Por exemplo, este código JavaScript define um texto grande e vermelho dentro de um rótulo com o ID "meurotulo":

$("#meurotulo").find("p.geformdin-controle-rotulo").find("span").html("<big><font color=red>Warning:</font> Use at your own risk.</big>");
