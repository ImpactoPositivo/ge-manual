Visão geral

Um Formulário Dinâmico, é uma interface intuitiva de arrastar e soltar usada para projetar formulários eletrônicos para os usuários finais preencherem. Um Formulário Dinâmico funciona como uma tela onde vários controles, como botões, caixas de texto, menus suspensos, etc., podem ser facilmente colocados para criar um formulário atraente e legível que pode ser salvo, duplicado ou impresso com o clique de um botão.

Esta página fornece informações sobre como criar, editar e excluir um Formulário Dinâmico, bem como informações sobre como um Formulário Dinâmico responsivo será exibido em três dispositivos diferentes.

Criando formulários dinâmicos

Para criar um novo Formulário Dinâmico, vá para a opção Formulário Dinâmicos no menu Objetos do Processo na aba Modelagem. Mantenha o ponteiro do mouse sobre o ícone de mais (+) e clique em Criar:

Uma janela modal será exibida contendo as opções para criar um Formulário Dinâmico do zero.

  1. Título: (Campo obrigatório). Insira o título do Formulário Dinâmico neste campo. Quanto mais descritivo for o título, mais fácil será entender do que se trata. O título do Formulário Dinâmico deve ser único (significa que dois ou mais Formulários Dinâmicos não podem ter o mesmo título no mesmo projeto).

    Observação: Leve em consideração que é possível criar Formulário Dinâmicos utilizando apenas um caractere em seu título.

  2. Descrição: Insira uma descrição para o Formulário Dinâmico. A descrição não será vista por usuários comuns de produção que executam casos, portanto, complete a descrição para os desenhistas de processo que possam precisar editar o Formulário Dinâmico posteriormente. Como a descrição não quebra a linha automaticamente, use a tecla "Enter".
  3. Cancelar: Cancela a criação do Formulário Dinâmico e fecha a janela modal. Também pergunta ao usuário se deseja cancelar e descartar todas as alterações feitas.
  4. Salvar e Abrir: Salva a configuração feita sobre o novo Formulário Dinâmico e abre o editor HTML5/Responsive Formulário Dinâmicos.
  5. Salvar: Salva a configuração feita sobre o novo Formulário Dinâmico. Lembre-se que é obrigatório o preenchimento de todos os campos obrigatórios, caso contrário a configuração não será salva e os campos obrigatórios que não foram preenchidos serão destacados em vermelho. Quando a configuração do novo Formulário Dinâmico é salva, uma mensagem flash aparece na parte superior da janela indicando que o novo Formulário Dinâmico foi salvo com sucesso.

HTML5/Formulário Dinâmicos Responsivos

O Gestor Express aprimorou a ferramenta de desenho do Formulário Dinâmicos, tornando-o muito mais fácil de usar. Essa ferramenta oferece aos usuários uma interface para criar um Formulário Dinâmico de 12 colunas que também cumpre a promessa HTML5 "gravar somente uma vez, executar em qualquer lugar". Esses Formulários Dinâmicos se ajustam automaticamente em diferentes tamanhos de tela, como desktops, tablets e telas de dispositivos móveis. Além disso, essa ferramenta de desenho é baseada em variáveis, o que significa que as variáveis já são criadas em uma interface separada e prontas para uso dentro de Formulários Dinâmicos responsivos.

Observação: É possível criar variáveis da ferramenta de desenho de Formulários Dinâmicos utilizando a propriedade Variavel nos controles da aba modelagem.

O menu modelagem também tem a opção Visualizar que exibe a visualização de como o Formulário Dinâmico será exibido ao executar casos. Aqui o desenhista de processo pode experimentar o formulário para ver como os objetos no formulário aparecerão e como eles serão preenchidos com valores. O modo de visualização pode ter pequenas diferenças de quando é executado em um caso, especialmente se ele usa campos que dependem de valores inseridos em formulários dinâmicos anteriores ou variáveis de caso definidas em gatilhos anteriores.

Visualização do tablet:

Visualização de smartphone

Observação: Leve em consideração que a visualização mostrada no navegador da web é diferente do aplicativo Gestor Express Mobile para Android e IOS que renderiza os controles Formulário Dinâmico modificados devido a problemas de compatibilidade com webkits e usabilidade em um espaço reduzido.

É assim que fica no smartphone real.

Observação: Leve em consideração que em Smartphones não é possível criar Formulário Dinâmicos. Eles são renderizados apenas ao usar o aplicativo.

Nenhum desses Formulário Dinâmicos precisa ser reconstruído, depois de criados dentro do designer, eles se ajustarão automaticamente para serem exibidos em desktops, tablets e smartphones.

Gerenciamento do Formulário Dinâmico

Para ver a lista de Formulário Dinâmicos criados dentro de um projeto, clique em "Formulário Dinâmicos":

Veja a lista de Formulário Dinâmicos com suas respectivas opções:

  1. Pesquisar:esquisa o texto introduzido na lista Formulário Dinâmicos. Todas as coincidências nos títulos das Formulário Dinâmicos serão filtradas à medida que o texto for inserido.
  2. Criar: Abre a janela Creation Blank Formulário Dinâmico para criar um novo Formulário Dinâmico. É possível criar Formulário Dinâmicos usando esta opção ou esta opção.
  3. Mostrar ID: Exibe o ID exclusivo do Formulário Dinâmico selecionado.
  4. Título: Mostra o título de cada Formulário Dinâmico. A lista mostra apenas os primeiros 10 elementos antes de criar uma nova página na janela. A lista é classificada em ordem crescente. Para alterar a ordem da lista para ordem crescente, clique no triângulo para baixo no rótulo da coluna.
  5. Editar: Abre o editor Formulário Dinâmico.
  6. Excluir: Exclui o formulário dinâmico. Uma janela de confirmação será exibida:

    Clique em Sim para excluir o formulário dinâmico ou clique em Não para fechar a janela sem excluir o formulário. Quando um Formulário Dinâmico é excluído, uma mensagem flash é exibida na parte superior da janela indicando que o Formulário Dinâmico foi excluído com sucesso.

    Não é possível excluir um formulário dinâmico que foi atribuído a uma tarefa, se for o caso, uma mensagem de aviso será exibida.

    Ele mostrará quantas dependências o Formulário Dinâmico possui. As possíveis dependências de um Formulário Dinâmico são: tarefas, supervisores, permissõess e Rastreador de Casos. Certifique-se de que o Formulário Dinâmico não tenha dependências antes de excluí-lo.

  7. Paginação: Quando mais de 10 Formulários Dinâmicos são listados, uma nova página é criada. Para navegar pelas páginas use as opções > ou < ou clique sobre o número da página para procurar um Formulário Dinâmico.
